AI Telegram Bot - установка (Railway)

1) Подготовьте переменные окружения:
- TELEGRAM_TOKEN
- CRYPTO_BOT_TOKEN (см. @CryptoBot в Telegram → Crypto Pay / приложение)
- ADMIN_IDS (пример: 123456789)
- ADMIN_USERNAME (пример: @my_username)

   Для чата с AI — один из вариантов:
   а) DEEPSEEK_API_KEY — ключ DeepSeek (sk-...). Взять: https://platform.deepseek.com → API Keys (бесплатные лимиты есть).
   б) API_BEARER_TOKEN — ключ от сервиса onlysq.ru (если он у вас есть).

2) Установите зависимости:
pip install -r requirements.txt

3) Локальный запуск:
python bot.py

4) Деплой на Railway:
- Подключите репозиторий
- Build Command: pip install -r requirements.txt
- Start Command: python bot.py
- Добавьте все переменные окружения из п.1

5) Безопасность:
- Не храните токены в коде/репозитории
- Никогда не коммитьте `.env`
- Ограничьте список `ADMIN_IDS` только своими ID
- Регулярно ротируйте токены
